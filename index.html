Meus Flashcards de Estudos

O que é HTML?

É uma linguagem de marcação utilizada na construção de páginas na Web.

O que é CSS?

É uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML.

Para que serve o Flexbox no CSS?

Para alinhar e distribuir o espaço entre os elementos de uma página, de forma flexível e responsiva.

O que faz a propriedade 'perspective'?

Ela define a distância entre o utilizador e o plano Z, criando o efeito de profundidade 3D no elemento.
